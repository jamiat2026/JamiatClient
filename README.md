# Jamiat Admin

Jamiat Admin is a web dashboard for managing users and donation workflows, built with Next.js (App Router).

## Features
- Admin authentication
- Invite-based onboarding
- Donation certificate generation and email delivery

## Getting Started

Install dependencies and run the dev server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open `http://localhost:3000` in your browser.

## Project Structure

- `app/` Next.js routes and API handlers
- `lib/` shared utilities
- `public/` static assets

## Configuration

Environment variables are stored in `.env`.
